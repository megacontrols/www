Mega Controls Logo Font ver.3.00 How to install and use the Logo Font?
To install, go to Control PanelFontsFileInstall New Font... and locate the unzipped font file named Mega.ttf and click "OK".
To use, open any applicable program and choose "Mega" from the font list. Next, type "M" or "m" to display Mega, type "C" or "c" to display Controls or type "G" or "g" to display Mega Controls in the logo font.
